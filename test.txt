じゃんけんの手の出し方
条件：N = p + c + g, M = 5*p + 2*c
全勝する時の指の本数をMXとする。
M-MX＝Xを求める。
手を変えるときの差分を以下のように書く。
p → g = Fpg = -5, Fpc = -3, Fcg = -2, Fgc = 2, Fcp = 3, Fgp = 5
条件：N = px + cx + gx, MX = 5*px + 2*cx + 0*gx　を満たす(px,cx,gx)
の組は1つである。
px * Fpg + cx * Fcg <= X <= gx * Fgp + cx * Fcp

X > 0　のとき、
X = a * Fgp + b * Fcp + c * Fgc (a + c <= gx, b <= cx)
X < 0　のとき、
X = a * Fpg + b * Fcg + c * Fpc (a + c <= px, b <= cx)　
(a,b,c)の組のうち、　a + b + c = L　が最小となる（Lmin）ものを求める。

方針：
出来るだけaを大きくし、数合わせにbとcを使う。aが不足する場合はbを使い、
数合わせにcを使う。
